# 着信を拒否したい

[`<Reject>`&nbsp;TwiML動詞](https://www.twilio.com/docs/voice/twiml/reject)を使用し、着信を拒否するようにプログラムできます。さらに、発信者のエクスペリエンスを変更する理由も設定できます。`reason`属性を`"busy"`に設定すると、ビジー信号を発生させることができます。`reason`を空白にするか、デフォルトの`"rejected"`に特に設定すると、標準である「現在使用されていません」タイプの応答が行われます。

`<Reject>`(拒否)された通話に料金を支払う必要はありません。そのためこれは迷惑電話に対し、効果的です。

この宝箱をこじ開けるには、特定の電話番号をブロックします。まず、コードにより拒否される対象となる番号を決めます。次に、その番号を条件付きでブロックするコードを記述します。テストの終了後、その番号を**HACK**インターフェイスに入力し、経験値と素晴らしいアイテムを獲得しましょう!
